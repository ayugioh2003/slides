## 2-1 收錄的形式

* 視覺
* 聽覺
* 語意

note:

語意: 草莓、蕃茄

--

收錄形式的記憶表現比較

|          | 記憶表現的比較     | 相關資料          |
| -------- | ------------------ | ----------------- |
| 短期儲存 | **聽覺** > 語意 > 視覺 | <small>(Baddely, 1966)</small>   |
| 長期儲存 | **語意** > 視覺、聽覺  | <small>(Vousfield, 1953)</small> |

--

2-1 收錄的形式 小結

* 短時間內快速記住 → <b>聽覺</b> <!-- .element: class="fragment "--> 
* 長時間持續記住 → <b>語意</b> <!-- .element: class="fragment "-->

## 2-2 訊息的轉移

* 訊息從短期記憶轉移到長期記憶<!-- .element: class="fragment " data-fragment-index="1"--> 
* 固化：刻意專注於訊息並理解它<!-- .element: class="fragment " data-fragment-index="2"--> 
* 複誦、記憶術技巧<!-- .element: class="fragment " data-fragment-index="3"--> 

note:
將新訊息與已知訊息，彼此建立連結或聯想

### 2-2-1 複誦

* 練習效果：重複複誦某個項目<!-- .element: class="fragment " --> 
* 最佳的複誦新訊息的時間安排為何？<!-- .element: class="fragment " --> 
* 分散練習 vs 密集練習 (Ebbinghaus, 1885)<!-- .element: class="fragment " --> 
* 白話：定時複習 vs 臨時抱佛腳<!-- .element: class="fragment " --> 

--

為什麼分散練習效果好?

1. 在不同脈絡下學習，分歧脈絡有助於訊息加強固化
2. 睡眠影響記憶: 睡眠時，速眼動時期的的量 <small>(Smith, 1996)</small>

為了讓複誦的訊息進入長期記憶

* 單純複誦效果不大 (維持性複誦)
* 我們需要有策略的複誦 (精緻性複誦)


note:
除了固化，還要談再固化

* 記憶被提取後，會變得不穩固
* 為了防止記憶被衰退、破壞，需要再固化
* 因此要避免單調的維持性複誦
* 需要精緻性複誦
